# 🌿 Plant Disease Detector

A deep learning-based web application for identifying plant diseases from leaf images. The model has been primarily trained and optimized on strawberry and tomato plant diseases, while also demonstrating the ability to recognize patterns in other plant leaf images.

## Key Features

* Plant disease detection from leaf images
* CNN-based image classification
* Grad-CAM visualizations for explainable AI
* Interactive web interface
* Deployed on Hugging Face Spaces

## Model Scope

The model is primarily trained on:

* Tomato diseases
* Strawberry diseases

While the application may provide predictions for other plant leaf images, the highest accuracy and reliability are expected for strawberry and tomato plants.
